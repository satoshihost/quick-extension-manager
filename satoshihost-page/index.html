How It Works

Simple and intuitive controls at your fingertips

1

Search

Type in the search box to filter extensions by name or description

2

Toggle Switches

  • Blue: Extension is enabled
  • Gray: Extension is disabled
  • Disabled System extensions
3

Visual Indicators

  • Normal : Extension is enabled
  • Faded: Extension is disabled
  • Yellow: cannot be disabled
4

Stats

The bottom shows "X/Y extensions enabled," where X is the enabled count and Y is the total count
